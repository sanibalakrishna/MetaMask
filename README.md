## Setup Locally
Clone this repo using below command
```bash 
git clone https://github.com/sanibalakrishna/MetaMask.git 
```
Install the dependencies in terminal pointing to root folder
```bash
npm install
```
Finally to run server
```bash
npm run dev
```
